# 作者按

（以下内容原载于本人微信公众号）

大家好，这里是爱飞 744 的城管二人组[1]。 好久不见，甚是想念。距离本公众号的上次更新已经有四个月之久了。不知道大家还记不记得我？当然，有些人已经把我忘掉了。

但没有关系，本公众号经历了四个月之久的整顿，现在已经准备整装待发了。 这次给大家带来的，就是叫做《旭日东升》（Rise from East，RFE）[2]的小说系列。这个系列我构思已久，从查资料，打草稿，然后再逐渐成为一个能看得过去的雏形。

目前这个小说的写作尚未完成，而且形式颇具有电影行业中的蒙太奇手法的风格。所以当你第一次阅读这个作品的时候，可能会难懂 （主要是因为写得太烂了）。

说实话，这个小说就是把《魔法禁书目录》的世界观加上真实世界中的历史相结合，从而写出来这么一个不明所以的玩意，所以某种程度上来说算是二创了。 小说的正文将会在我的个人博客（https://gatg.top） 上发布，公众号上关于小说的内容仅为本小说系列的相关故事解读以及幕后写作故事。 那，我们就一起进入一个光怪陆离的世界吧（笑）。

# 简介

时值公历 2012 年龙年，全世界在赤色巨龙的包围下生存着。

冷战的氤氲消散，却留下了一个千疮百孔的世界。从欧罗巴到美利坚，从亚细亚到阿非利加，各地的政权纷纷更迭，秩序崩塌，曾经的强国风光不再，文明的光辉也黯然失色。

在古老的东方大陆的孕育下，中华文明在寰宇世界中历经上下五千年，从屈辱的晚清，再到辛亥革命、北伐战争，最终迎来了 1949 年的解放。这个新兴国家的领导人们夙兴夜寐，通过一系列冷静而精妙的战略操作，这头巨龙将美利坚和苏联两大超级大国拖入了漫长的消耗战中。苏联在与中国的外交和经济对抗中逐渐失去了昔日的强盛，阿富汗战争昭示了苏联的颓败，八一九事件给她钉上了棺材板的最后一颗螺钉，最终在 1991 年四分五裂。美利坚在越战的泥沼中无法自拔，于八九十年代的石油危机、次贷危机中轰然崩塌，最终在末任且短命的老布什总统宣告之下化为三国，群雄逐鹿。她的盟友们虽然未至此境，但在面对日益强大的中国时，已不再是那个团结一心无可匹敌的北约。至此，中国凭借着自身的力量，成功从美苏冷战中熬出，将两个世界大国推入了崩溃的深渊。如今，这头苏醒的巨龙正以无法阻挡的气势腾飞于九霄之上。她的经济奇迹如滔滔江水般席卷全球，她的科技进步如雷霆万钧般震撼世界。

然而，伴随着中华复兴的是东亚的动荡与不安。隔海相望，一个同样充满野心的存在 —— 学园都市，正悄然崛起。二战后，亚雷斯塔・克劳利铸造的这座钢筋水泥之城，代表着科技的巅峰，掌控着全世界最先进的超能力研究和武器开发。但他们也在紧盯着中国的一举一动。随着中国在科技和军事上的日益强大，学园都市的领导者们愈发感到危机四伏。两大强权之间的关系变得愈加紧张，每一次技术突破、每一个军事动作都可能成为点燃战争的导火索。而在她的阴影下，曾经的帝国主义残余依然在暗中伺机而动。为了维持其全球领先的科技地位，他们加快了超能力者的培养和科研投入，并在太空和深海等新兴领域展开了多方面的探索。更为重要的是，学园都市的领导者意识到，仅凭科技优势已经无法确保未来的安全，他们开始寻求与日本建立更紧密的合作，以形成对中国的战略包围。

两大强权之间的对抗日渐激烈，表面上的和平不过是即将爆发的风暴前的宁静。无数次秘密交锋和明争暗斗，使得双方都疲惫不堪。然而，真正的冲突从未远去，它像一颗定时炸弹，随时可能将整个世界卷入火海。这种你追我赶的局势，让整个世界都感受到了前所未有的紧张气氛。国际社会陷入了新一轮的军备竞赛，各国纷纷站队，形成了以中国为中心和以学园都市为中心的两大阵营。经济制裁、科技封锁、舆论战等手段此起彼伏，每一场国际会议都暗藏玄机，每一次外交交锋都如履薄冰。战争的阴影越来越浓烈，而和平的曙光却似乎遥不可及。

在这样的背景下，一次意外的泄漏事件彻底打破了这微妙的平衡。学园都市与中国之间的紧张关系陡然升级，从原本的冷战状态转为互不相让的剑拔弩张。两国的人民都在屏息等待，仿佛一触即发的战争已经在眼前。两国的外交使节来往频繁，然而每一次谈判都以更大的不信任和敌意告终。军队在边境线集结，战机在夜空中呼啸，仿佛一触即发的战争已经在眼前。

而在这复杂的国际局势下，决定未来走向的关键人物，不是那些坐在高层会议室中的政客，而是一个身处学园都市的普通高中生 —— 上条当麻。他手中的选择，不仅关乎学园都市的存亡，也将决定东亚乃至全世界的命运。

此刻，他站在历史的十字路口，面对着一个无法回避的抉择：是选择退让，换取短暂的和平？还是选择迎战，以血火铸就未来？无论如何，这场即将到来的风暴，将彻底改变这个世界的格局。上条当麻并不愿意成为英雄，他渴望的只是一个平静的生活，然而命运却没有给他选择的余地。在这个关乎全人类未来的十字路口，他必须作出决定，而他的决定将注定载入史册，无论是光辉灿烂的篇章，还是满布血腥的悲剧。

# 中国

在古老的东方大陆上，中华民族的智慧与力量孕育了五千年的文明。从长江黄河的浩荡到万里长城的雄壮，这片土地承载了无数代人的梦想与奋斗。历经战火洗礼与风雨飘摇，中华大地上的人民从未屈服，始终顽强地守护着他们的家园与信仰。然而，曾经辉煌的帝国也曾陷入沉寂，百年屈辱令这个古老的文明一度失去了它的光彩。

自 1840 年鸦片战争以来，中国的命运仿佛陷入了无尽的黑暗。列强环伺，内忧外患，晚清政府的腐朽无能使得整个国家陷入了前所未有的危机。随着西方列强的侵略与掠夺，中华民族的尊严被一次次践踏。辛亥革命推翻了两千多年的封建帝制，却未能带来真正的强盛。军阀混战、民不聊生，中国仿佛在风雨中飘摇的一叶孤舟，随时可能沉没。

然而，在这片古老的土地上，从未缺少改变命运的勇气与决心。1949 年，毛泽东在天安门城楼上庄严宣告：“中华人民共和国中央人民政府今天成立了！” 这句话如雷霆般响彻神州大地，也宣告了一个新时代的到来。自此，中国人民在中国共产党的领导下，开始了一场前所未有的伟大复兴。

在领导人们的夙兴夜寐和改革开放的旗帜下，中国以惊人的速度重新崛起。原本一穷二白的国家，在短短几十年间，变成了世界第一大经济体及第一大国，人民的生活水平显著提高。高楼大厦在中国的大地上如雨后春笋般拔地而起，先进的基础设施、现代化的工厂与实验室遍布全国。中国不再是那个被动挨打的弱国，而是一头真正的东方巨龙，屹立在世界舞台的中央。

然而，中国的崛起并非一帆风顺。为了确保国家安全与战略利益，中国在面对国际挑战时展现出了坚韧不拔的意志和卓越的战略智慧。在冷战风云密布的几十年里，中国逐步将美苏两大超级大国拖入了一场旷日持久的竞争与对抗。通过灵活的外交手段和冷静的战略决策，中国成功化解了多次国际危机，迫使苏联在经济和政治压力下最终解体，而美国则陷入了长期的国内外困境，逐渐丧失了全球霸主的地位。

中国的崛起不仅仅体现在经济的飞速增长，更在于其卓越的科技创新能力。依托于国家强大的政策支持和庞大的科研投入，中国在信息技术、人工智能、量子计算、基因工程等诸多领域取得了举世瞩目的成就。中国的航天事业更是迎来了前所未有的辉煌，嫦娥探月、天宫空间站的建设让中国在太空探索领域独占鳌头。与此同时，全球瞩目的 “一带一路” 倡议则将中国的影响力扩展至全球，从亚非大陆到欧洲，从拉丁美洲到大洋洲，无处不见中国的身影。

国家的强大不仅依赖于经济和科技，更需要强大的军事实力作为后盾。中国人民解放军在现代化进程中实现了脱胎换骨的蜕变，从传统的机械化部队转变为一支信息化、智能化的现代化军队。中国的核威慑力量和常规军事力量也随着时间的推移不断增强，为国家安全筑起了坚不可摧的盾牌。在东亚地区，中国通过军事演习和战略部署，稳步推进对南海和东海的掌控，进一步巩固了自己的地缘政治优势。

面对学园都市的迅速崛起，中国政府意识到，这不仅仅是一次普通的国际竞争，而是关乎国家生死存亡的战略挑战。学园都市凭借其超能力技术在全球范围内迅速扩展，已成为世界上唯一能够与中国相抗衡的力量。然而，中国并未因此退缩，而是以更大的决心和毅力推进自身的科技创新和军事现代化进程。中国的科学家们夜以继日地工作，努力在超能力研究领域取得突破，以应对学园都市的挑战。

与此同时，中国在国际舞台上也采取了更加积极的姿态，寻求与其他国家建立更紧密的合作关系。通过巧妙的外交布局，中国成功拉拢了一批对学园都市心存疑虑的国家，形成了一个广泛的国际联盟，共同对抗学园都市的扩张势力。这一战略举措不仅增强了中国的国际影响力，也进一步孤立了学园都市，使其在国际社会中陷入了前所未有的困境。

这片古老而又焕发新生的土地，承载着亿万人民的梦想与希望，承受着亿万人民跳动着的心脏。无论未来多么艰难，中国都将继续前行，在世界舞台上绽放出属于自己的光辉。在这个日益复杂的世界中，中国正以不屈的姿态，迎接每一个挑战，书写属于中华民族的新篇章。无论前方的路有多么艰险，这头东方巨龙都将以她不屈的意志和强大的力量，在这场全球博弈中争得属于自己的胜利。

# 学园都市

学园都市，这座位于东京西部的庞大科技都市，早已不仅仅是一个教育与研究中心，它是人类科技与未来的象征。在这个被高墙包围的城市内，最前沿的科学研究与最神秘的超能力开发正在进行，数万名天才学生在这里接受严格的训练，他们的潜力被分为六个等级，从最基础的 “无能 (Level) 力 (0)” 到令人瞠目结舌的 “超能 (Level) 力 (5)”。每一个在学园都市接受教育的学生，都是人类未来的希望，亦或是潜在的威胁。

学园都市的建筑群高耸入云，每一栋大楼内都隐藏着无数的秘密实验室，科研人员们夜以继日地工作着，他们的任务不仅仅是推动科技的发展，更是在学园都市的领导者的严密监控下，打造出一支前所未有的超能力者军队。这支军队被认为是学园都市立于世界巅峰的终极王牌。其背后所蕴藏的力量足以令任何一个国家为之胆寒。无论是量子物理、生物工程，还是时间空间的操控，学园都市都走在了世界的最前沿。

然而，学园都市并非铁板一块。虽然表面上是一个高效运转的科技乌托邦，但在其内部，暗潮涌动。各个派系的权力斗争从未停止，科研机构之间的竞争愈演愈烈，学生群体的抗议活动时有发生。尤其是在拥有 “超能力” 的学生中，那些被评为 “无能力” 的人往往倍感挫败，他们的愤怒与不满如同火山般积蓄，随时可能爆发。而那些被称为 “超能力者” 的学生，则处于另一种极端的危险之中。他们的力量越大，被监控的程度就越高，他们的一举一动都在被权力者们密切关注着。稍有不慎，他们可能就会从学园都市的明星变成被清除的目标。

作为学园都市的实际统治者，统括理事会掌握着生杀大权。他们的决定影响着数万名学生和无数科研人员的命运。每一个新项目的启动，每一项研究成果的公布，都必须经过他们的批准。他们对外界封锁一切信息，学园都市的一切对外事务都由他们控制。尽管他们宣称一切都是为了科学的进步和人类的未来，但学园都市内外对他们的质疑声却从未平息。

学园都市与外界的关系也异常复杂。作为一个独立的实体，学园都市拥有高度的自治权，不受日本政府的直接控制。然而，随着中国的崛起以及其在全球范围内的影响力不断扩大，学园都市开始感受到来自东方的巨大压力。为了应对这一挑战，学园都市加快了其军事化进程，不仅大幅增加了对超能力者的培养力度，还秘密开展了多项军事科研项目，试图在未来的科技战争中取得决定性优势。

尽管如此，学园都市并不孤立。它与世界上许多国家和组织都保持着复杂的关系。许多国家渴望获得学园都市的科研成果和技术支持，而学园都市也通过这些合作，在全球范围内扩展其影响力。尤其是在西方国家，学园都市的技术输出已经成为这些国家维护自身科技优势的重要途径。然而，这种合作也使得学园都市成为了国际间谍活动的焦点，各国情报机构在这里展开了激烈的角逐，试图获取学园都市最核心的机密。

然而，所有这些力量的角逐和科技的进步背后，学园都市却未能逃脱自身发展中的内在矛盾。超能力的开发在不断突破人类认知的界限，也在不断挑战着社会的伦理和法律底线。越来越多的人开始质疑，学园都市所追求的终极目标，究竟是为了人类的进步，还是仅仅为了满足少数人对权力和控制的欲望。

更为严重的是，学园都市内部开始出现一些反对现有秩序的力量。这些人认为都市的科技发展已经偏离了最初的目标，过于追求力量而忽视了人性的光辉。他们反对以牺牲多数人为代价的超能力研究，主张通过和平与合作来解决问题。一些不满现状的科学家和政客开始考虑与中国合作，甚至背叛学园都市的统治阶层。这些反对派虽然势单力薄，但他们的声音却引起了不少学生和普通市民的共鸣。逐渐地，这股力量在暗中积聚，试图寻找机会推翻现有的权力结构，让学园都市的未来变得更加不确定。

学园都市的高层对此心知肚明，他们知道，一旦内部反对势力与外部强权联手，这座科技之城将面临前所未有的危机。为了应对这一潜在的威胁，高层开始采取更加严格的监控措施，试图将一切反抗的苗头扼杀在萌芽状态。然而，这样的压制却进一步激化了内部矛盾，反对派的行动变得愈发激烈与隐秘。

更为恼火的是，学园都市与中国之间的对抗愈演愈烈。都市高层明白，保持科技领先是他们唯一的立身之本，但面对中国这个庞然大物的步步紧逼，他们也感到前所未有的压力。都市的科研人员加紧开发新型超能力技术，试图在这场对抗中占据优势；而军方则不断增强防御力量，防备中国可能发起的任何攻击。

然而，无论是内部的纷争，还是外部的压力，学园都市都已经来到了一个岔路口。它将继续维持其科技霸主的地位，还是在内外交困中逐渐走向衰落？未来的命运悬而未决，而这座城市的每一个人，都将在这场即将到来的风暴中扮演重要的角色。

在这座科技都市的未来，既充满了光明与希望，也潜藏着不可预见的黑暗与危机。学园都市是一个奇迹，但它也是一个随时可能崩塌的庞然大物。在这座城市的命运中，未来的历史或将被重新书写，世界的格局也可能因此彻底改变。

# 全球

当 2012 年龙年的钟声敲响时，整个世界已然在钢丝绳上踱步着进入了一个新的时代。这个时代充满了不确定性与潜在的冲突，全球的政治版图正在经历剧烈的震荡和重组。在这个动荡不安的世界中，各大洲的命运都在悄然发生着剧变。冷战的结束并未带来期盼已久的和平与繁荣，反而在旧有的秩序崩溃后，留下了一个千疮百孔的地球。从欧罗巴的瓦砾堆到美利坚的颓废都市，从亚细亚的动荡边陲到阿非利加的饱经风霜，整个世界似乎都在为生存而挣扎。

欧罗巴，曾经的文明摇篮，如今却处于深重的危机之中。几十年前的战火与硝烟已经远去，但留下的疮痍与创伤却始终无法愈合。欧盟曾试图通过经济与政治的统一来重建昔日的辉煌，然而内部的纷争与外部的压力使得这个共同体摇摇欲坠。各国之间的分歧不断加深，右翼与左翼的对立愈发激烈，社会动荡成为了欧罗巴大陆的常态。南欧的债务危机、西欧的宗教危机和东欧的难民潮使得这个曾经以统一与繁荣为目标的联盟逐渐分裂。昔日的联盟余晖早已散尽，取而代之的是一片暮气沉沉的景象。

与此同时，大西洋彼岸的美利坚合众国，这个曾经的全球霸主，也陷入了前所未有的内忧外患之中。三国内战、经济增长停滞、贫富差距扩大、社会阶层分化，这一切都使得美国的内政愈发不稳定。长达数十年的对外战争与金融危机耗尽了这个国家的元气，国内的政治极化与社会撕裂使得曾经的民主灯塔失去了往日的光辉。各州之间的矛盾加剧，联邦政府的权威不断受到挑战，曾经牢不可破的国家纽带正面临前所未有的考验。外部，中国和学园都市的崛起更让美利坚感受到了难以言喻的压力，昔日的世界警察如今不得不面对一个更加复杂而难以控制的国际局势。

在亚非拉地区，国家间的竞争与合作交织在一起，资源争夺、民族冲突与全球化的浪潮使得这些地区变得更加复杂。南亚和东南亚国家在中国的影响下，逐渐融入了一个以东方为中心的新经济圈，然而，这些地区的发展同样伴随着严重的社会问题和环境危机。学园都市，作为这片大陆上的另一股强大力量，也在悄然崛起，其对超能力的研究与应用使得亚细亚的科技竞争变得更加激烈。

在非洲与中东地区，战乱与贫困依旧困扰着这片资源丰富却命运多舛的土地。大国争相将其作为资源争夺与战略布局的焦点，然而这片大陆上的人民却在战争与贫困中苦苦挣扎。新一轮的殖民主义以更加隐蔽的形式回归，各国之间的矛盾与冲突层出不穷，跨国武装冲突、内战、恐怖主义等问题让整个地区陷入了无尽的混乱之中。尽管一些国家试图通过现代化建设与经济改革来摆脱过去的桎梏，但内战、腐败与国际干预的阴影始终挥之不去，使得整个大陆笼罩在一片不祥的气息中。各国人民为了生存，不得不在战火与动荡中挣扎，整个地区似乎正滑向无法控制的深渊。

学园都市和中国的科技竞赛已不再是单纯的技术较量，而是一场关系到全球权力格局的战略博弈。各大国也在这场竞赛中选择站队，形成了以中国为中心和以学园都市为中心的两大对立阵营。世界各国的领导者们都在密切关注着这场竞赛的动向，每一个决定、每一个行动，都可能引发连锁反应，将全球推向新的战争深渊。经济制裁、科技封锁、代理人战争等手段不断升级，各种明争暗斗层出不穷。曾经的盟友反目成仇，新的同盟在利益驱使下暂时结成，但这一切都如同悬在头顶的达摩克利斯之剑，随时可能坠落。

而在这复杂的国际局势下，全球的普通民众也感受到了深深的不安与焦虑。人们担心战争的阴影再度降临，担心自己的家园成为大国角力的牺牲品。各国政府为了维护自己的利益，不断加紧军备，推进国防现代化，国际军备竞赛愈演愈烈。整个世界都仿佛置身于一场无休止的恶性循环之中，和平似乎越来越遥不可及。

在这种全球性动荡的背景下，各国之间的对立与冲突愈发尖锐。曾经的国际合作与多边主义逐渐被孤立主义与大国竞争所取代，全球化的成果也在迅速消解。经济危机频发，环境恶化加剧，各地的民众生活在不断加深的不安与恐惧中。资源的争夺、科技的竞赛、军事的对抗，所有这些因素交织在一起，使得世界变得更加脆弱而危险。

在混沌的千禧十年之后，是未知的未来。可究竟是和平的曙光，还是战争的烈火？没有人知道。唯一可以确定的是，无论是在学园都市的地下反抗组织，抑或是美利坚的民权运动，世界各地的人们都在用自己的方式对抗这即将吞噬一切的黑暗。“世界将何去何从？”，这是每一个国家、每一个领导人、每一个普通人都在思考的问题。

# 如果你是原著党的话......

这个小说对原作的世界观进行了非常大的改动，甚至可能是另一个平行世界，你可能不太适应。如果你对此非常介意，我的建议是最好不看，毕竟眼不见心不烦。

雷点包括但不限于：

  1. 魔法侧势力几乎全灭，除了茵蒂克丝，其他魔法侧的人几乎只是走个过场，且不对剧情产生影响。
  2. 中国的科技实力与学园都市不分上下，甚至在某方面领先于学园都市。
  3. 几乎没有新约和创约的内容。
  4. 含有军事方面的相关描写，可能需要具备一些军事知识。
  5. etc.....
